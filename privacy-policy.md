Privacy Policy – MBTA Widgets
Effective Date: 04/23/2026
MBTA Widgets respects your privacy. This Privacy Policy explains how we collect, use, and protect information when you use the app.
 
1. Information We Collect
We collect limited, non-personal, anonymous data to improve app performance, reliability, and real-time features.
This includes:
•	A randomly generated anonymous device identifier (UUID)
•	MBTA usage data (routes, stops, directions viewed)
•	API performance data (response times, success/failure rates)
•	App usage timestamps
•	Widget refresh activity
For Live Activities (Dynamic Island / Lock Screen updates), we temporarily collect:
•	APNs push token (used only to deliver Live Activity updates)
•	Route ID and route name
•	Stop ID and stop name
•	Direction ID
•	Destination name
•	Tracked arrival time (timestamp)
•	Server timestamp of activity creation
 
2. Information We Do NOT Collect
We do not collect:
•	Name, email, or phone number
•	Location data
•	IP address
•	Payment information
•	Contacts, photos, or files
•	Any data that directly identifies you
 
3. How We Use Information
We use collected data to:
•	Deliver real-time transit updates via widgets and Live Activities
•	Monitor MBTA API performance and reliability
•	Identify and fix bugs
•	Improve app speed and stability
•	Understand general usage patterns (e.g., commonly used routes)
 
4. Data Storage and Processing
•	Anonymous usage data is stored securely using Supabase
•	Live Activity data (including push tokens and transit selections) is stored temporarily using Firebase
•	Push tokens are used only for Live Activity updates and are deactivated when no longer valid or when the activity ends
•	Data is not sold, rented, or shared for marketing purposes
•	Data is processed solely to operate and improve the app
 
5. Live Activity Processing
To provide real-time updates:
•	A secure server function periodically fetches transit predictions from the MBTA API
•	Updates are sent to your device via Apple Push Notification service (APNs)
•	This process uses temporary identifiers and does not track user identity
 
6. Legal Basis for Processing
We process anonymous usage data based on our legitimate interest in improving app functionality, performance, and user experience.
 
7. User Rights
•	No account is required to use the app
•	Data collection is anonymous
•	You can stop all data collection by uninstalling the app
•	Because data is anonymous, it cannot be linked back to an individual user
 
8. Data Retention
•	Anonymous usage data is retained for analytics and performance monitoring
•	Live Activity data (including push tokens) is temporary and removed or deactivated after the activity ends
•	No personally identifiable data is stored
 
9. Security
We use industry-standard security measures to protect stored data from unauthorized access, misuse, or disclosure.
 
10. Third-Party Services
We use the following external services:
•	MBTA API
Provides real-time transit data including arrival predictions, routes, and stops
•	Firebase
Used for temporary storage and processing of Live Activity updates
•	Supabase
Used for anonymous analytics and performance monitoring
•	Google Forms
Used for optional user feedback submission
•	Buy Me a Coffee
Used for optional user support and donations
 
11. Children’s Privacy
This app is not directed to children under the age of 13. We do not knowingly collect any personal data from children.
 
12. Changes to This Policy
We may update this Privacy Policy from time to time. Updates will be reflected by revising the effective date.
